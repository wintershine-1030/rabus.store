Store Image Changer Test

About this test:

This page demonstrates the Store Image Changer functionality. The script will automatically load images from the store/ folder and allow you to change them dynamically.

Keyboard Shortcuts:

  • → (Right Arrow): Next image (all images)
  • ← (Left Arrow): Previous image (all images)
  • R: Random image (all images)
  • S: Start slideshow (all images)
  • P: Change product images only

Product Image Controls:

Console Information:

Open your browser's developer console (F12) to see detailed information about image changes and available functions.

You can also use the global StoreImageChanger object in the console:

  • StoreImageChanger.getImages() - Get list of available images
  • StoreImageChanger.getCurrentIndex() - Get current image index
  • StoreImageChanger.changeImage() - Change to next image (all)
  • StoreImageChanger.setRandom() - Set random image (all)
  • StoreImageChanger.changeProducts() - Change product images only
  • StoreImageChanger.startProductSlideshow() - Start product slideshow

Product Image Testing:

This script now targets product images on product listing pages using these selectors:

  • [data-aid*="PRODUCT_IMAGE_RENDERED"] - Product image elements
  • [data-ux="Background"][role="img"] - Background image elements

On product pages, you can use the P key or StoreImageChanger.changeProducts() to change only product images.
